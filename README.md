# password-security-checker
Generador de contraseñas y validación con score de débil a fuerte


## 🔐 Password Security Checker

Aplicación en Python con interfaz gráfica que permite:

- Evaluar la seguridad de una contraseña
- Detectar contraseñas comunes
- Generar contraseñas seguras automáticamente
- Mostrar nivel de seguridad (débil a fuerte)
- Indicar qué elementos faltan (mayúsculas, números, etc.)

## 🛠 Tecnologías usadas
- Python
- Tkinter
